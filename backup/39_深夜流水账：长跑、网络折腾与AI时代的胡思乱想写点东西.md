## 周末的“网络考古”与 AI 工具横评

这周六日，我把大把时间耗在了折腾家里网络上，主要是为了彻底解决 **DNS 泄露**的问题。

折腾期间，我顺便做了一组“AI 盲测”——让 **DeepSeek、Gemini 和 ChatGPT** 去解决同一个网络坑。结果挺有意思，**ChatGPT 完胜**，剩下两个在复杂网络排错上还是差点意思。

最近我也开始尝试用 **Hermes** 来处理一些琐碎的自动化任务，不得不说，现在的 Agent 确实进化得越来越聪明了，常规问题基本能让人当个“甩手掌柜”。但作为技术牛马，**安全与权限分配**也是我最担忧的漏洞：

- **失去约束的风险**：没有严格限制的 AI 可以直接化身黑客攻击服务器。
- **隐私越界**：像 Hermes 这类 Agent，如果权限全开，它能肆无忌惮地读取和操作你本地的所有文件，甚至连网盘里的 iCloud 和 OneDrive 都能直接访问。

**我的解法是**：在安装和配置阶段必须严控权限。涉及敏感操作时，每一次下发指令都需要人工确认，并将其作为铁律固化到 Agent 的长效记忆当中。

## 关于“断更”两个月的周志反思

原本计划是每周写一次复盘的，今天去翻了一下 GitHub Issues 里的周志，才猛然发现自己居然已经**断更了快两个月**。

看来，习惯这东西确实没那么容易刻进DNA。为了拯救这个好习惯，我打算在提醒清单里加一条Tips：**每周日下午定时提醒**。本周末重新开张。
