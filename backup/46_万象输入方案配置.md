# [万象输入方案配置](https://github.com/coutureone/gitblog/issues/46)


### 1.配置文件

自定义配置文件新建` wanxiang.custom.yaml `

```yaml
# 万象拼音个人配置
# 用途：整理根目录 wanxiang.custom.yaml，适合 Mac 与 Hamster3 同步使用。
#
# 说明：
# - patch 下的配置会覆盖/补充 wanxiang.schema.yaml 的对应项。
# - 这份文件保留全拼、个人短语、候选数量三个稳定项。
# - 已开启造词功能：适合让常用新词逐渐变顺手。

patch:
  # 输入方案转写规则。
  # 当前使用全拼。以后如果输入 /zrm、/flypy、/mspy 等切换双拼，
  # 万象的 set_schema.lua 会改这里，所以这段应该放在根目录配置里。
  speller/algebra:
    __patch:
      # 可选模糊音示例：需要时取消注释。
      # - wanxiang_algebra:/模糊音_z_zh
      - wanxiang_algebra:/base/全拼

  # 个人快捷短语文件。
  # 对应根目录 custom_phrase_personal.txt。
  custom_phrase/user_dict: custom_phrase_personal

  # 候选数量。
  # 当前保留 6 个。万象注释里提示不要超过 6，
  # 因为 7/8/9/0 还会参与声调或其他输入逻辑。
  menu/page_size: 6

  # 造词功能。
  # recognizer/patterns/add_user_dict：允许用 `` 引导造词。
  # user_dict_set/enable_user_dict：启用已造词的用户词典读取。
  # add_user_dict/enable_user_dict：启用造词写入用户词典。
  # add_user_dict/enable_auto_phrase：开启自动无词频造词，让输入越用越顺。
  recognizer/patterns/add_user_dict: "^``[A-Za-z/`']*$"
  user_dict_set/enable_user_dict: true
  add_user_dict/enable_user_dict: true
  add_user_dict/enable_auto_phrase: true
```

新建配置个人自定义短语文件`custom_phrase_personal.txt`
