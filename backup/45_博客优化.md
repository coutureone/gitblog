# [博客优化](https://github.com/coutureone/gitblog/issues/45)



> 好久没有写博客了，自己的博客功能和显示还是有点问题，刚好借助AI给优化了一下

## 前端展示优化

- 修复首页社交图标下方横线间距过大的问题
  - 清掉 Mirror 原始图标 `margin-bottom` 带来的空白
  - 横线重新居中显示
- 优化字体策略
  - 改成系统字体优先
  - 不再优先使用 IBM Plex
  - 不引入远程字体/CDN，提升国内和海外访问速度
  - 代码块使用系统等宽字体
- 修复 Markdown 表格主题颜色 bug
  - 解决浅色模式下表格黑底、文字过暗的问题
  - 表格现在会跟随浅色/深色主题切换
  - 友链表格显示正常

## Mermaid 流程图优化

- 修复 Mermaid 渲染后仍显示 Loading 的问题
- 优化流程图字体显示，避免文字被遮挡
- 增加流程图自适应宽度
- 增加流程图工具栏
  - 放大
  - 缩小
  - 适应宽度
  - 复制源码
  - 全屏查看
- 支持点击流程图进入全屏查看

## 图片与资源优化

- 修正旧图片链接分支问题
  - 将 `gitblog@main` 兼容替换为 `gitblog@master`
  - 将 raw GitHub `main` 图片链接兼容替换为 `master`
- 给文章图片增加懒加载
  - `loading="lazy"`
  - `decoding="async"`
- 目标是通过 Typora + PicGo + GitHub Issues 写博客的流程
